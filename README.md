## 🌱 Pull Requests Rule

**Title**: N주차미션_닉네임 (ex. 1주차미션_제이)

**Reviewers(PR을 리뷰해 줄 팀원)**: 스터디원 (최소)1명

**Assignees(PR 담당자)**: 본인

**Labels**: mission

<br>

## ✔ Commit Message Convention

CHORE: 코드 수정, 내부 파일 수정

FEAT: 새로운 기능 추가

FIX: 버그, 오류 수정

DOCS: README 등의 문서 수정

REFACTOR: 전면 수정(코드 리펙토링)

TEST: 테스트 코드 추가 및 수정
